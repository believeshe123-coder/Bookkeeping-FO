# FutureFlow Economic Planner

FutureFlow is a lightweight browser-based cash-flow forecaster inspired by a personal Google Sheets workflow. It keeps the spreadsheet-like behavior—starting balance, recurring constants, one-off variables, running totals, and a trend chart—while presenting the experience as a focused financial planning application.

## Features

- Configure a starting balance, start date, and forecast end date.
- Add recurring constants for predictable items such as paychecks, rent, utilities, subscriptions, and weekly spending.
- Add one-time variables for irregular expenses or income.
- Automatically generate a chronological projection with running totals.
- View maximum and minimum projected balances plus a canvas-based balance chart.
- Automatically save your plan in the browser with an optional manual local save button.

## Run locally

Open `index.html` in any modern browser. No build step or external dependencies are required.
